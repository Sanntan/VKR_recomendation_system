# VKR Admin Panel (Vue 3 + Vite)

Локальная административная панель для работы с внутренним API проекта.

## Быстрый старт

```bash
cd admin-panel
npm install
npm run dev
```

По умолчанию панель запускается на [http://localhost:5173](http://localhost:5173).

### Переменные окружения

Можно задать базовый URL API через `.env`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

Если переменная не задана, используется `http://localhost:8000`.

## Основные возможности

- проверка статуса API;
- просмотр активных мероприятий и поиск по кластеру;
- поиск студента по `participant_id`, просмотр его информации;
- просмотр рекомендаций (с оценками score);
- ручной запуск пересчёта рекомендаций с указанием минимального score.

## Зависимости

- Vue 3 (Composition API)
- Vite
- Axios
- Pinia (как базовый store, резерв на будущее расширение)

## Структура

- `src/App.vue` — главная страница с основной логикой
- `src/services/api.js` — клиент работы с внутренним API
- `src/components/EventTable.vue` — таблица мероприятий/рекомендаций
- `src/styles/main.css` — базовые стили

## Примечания

- Для разработки API должен быть запущен и доступен (см. `src/core/config.py`).
- CORS уже разрешён для `http://localhost:5173`, при необходимости можно обновить список доменов в настройках сервиса.
